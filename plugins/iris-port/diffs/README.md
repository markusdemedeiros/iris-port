# diffs/ (empty scaffolding)

This directory holds **your** reviewed git diffs — the manual style cleanups a human
made to ported Lean files. The `/learn-diff` skill points the `lean-style-critic` at a
file here so it can generalize your corrections into house-style rules.

It ships empty on purpose. Drop diffs in as `diff_<topic>.txt`, then run:

    /learn-diff

and tell the assistant which file to study.
